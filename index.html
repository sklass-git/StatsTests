Assumption Check — Normality
normality
Shapiro-Wilk
Best for small samples (n < 50). Most powerful normality test.
W = (Σ aᵢxᵢ)² Σ (xᵢ − x̄)²
aᵢ = order statistic coefficients
xᵢ = ordered sample values
p < 0.05 → reject normality 
from scipy.stats import shapiro
import statsmodels.api as sm

print("Shapiro-Wilk Test for Normality (mNeonGreen/mCherry Ratio per Sample Type):\n")

alpha = 0.05 # Significance level

for sample_name in df_filtered['Sample_Name'].unique():
    sample_data = df_filtered[df_filtered['Sample_Name'] == sample_name]['Ratio']
    stat, p = shapiro(sample_data)
    print(f"Sample: {sample_name}")
    print(f"  Statistics=%.3f, p=%.3f" % (stat, p))
    if p > alpha:
        print(f"Data is Normally Distributed")
    else:
        print(f"Data is NOT Normally Distributed")
    print("\n")
Assumption Check — Variance Equality
variance
Levene's Test
Robust to non-normality. Preferred for real-world data.
W = (N−k) · Σnᵢ(Z̄ᵢ−Z̄)² (k−1) · ΣΣ(Zᵢⱼ−Z̄ᵢ)²
Zᵢⱼ = |xᵢⱼ − x̄ᵢ| (deviations)
k = number of groups
p < 0.05 → unequal variances 
from scipy.stats import levene

# Prepare the data for Levene's test
# We need a list of arrays, where each array contains the 'Ratio' values for a specific 'Sample_Name'
data_for_levene = [df_filtered[df_filtered['Sample_Name'] == name]['Ratio'].values
                   for name in df_filtered['Sample_Name'].unique()]

# Perform Levene's test
stat, p = levene(*data_for_levene)

print("Levene's Test for Homogeneity of Variances (mNeonGreen/mCherry Ratio per Sample Type):\n")
print(f"Statistics=%.3f, p=%.3f" % (stat, p))

alpha = 0.05 # Significance level

if p > alpha:
    print(f"Variances are equal across sample types")
else:
    print(f"Variances are NOT equal across sample types")
1 : 1  Pairwise — Parametric
parametric
Student's t-test
Equal variance, two independent groups
t = x̄₁ − x̄₂ sₚ · √(1/n₁ + 1/n₂)
sₚ = pooled std dev
x̄ = group mean
n = group size 
from scipy.stats import ttest_ind

# Extract the 'Ratio' data for 'Reporter_Norm_only' and 'Team1'
reporter_norm_only_data = df_filtered[df_filtered['Sample_Name'] == 'Reporter_Norm_only']['Ratio']
team1_data = df_filtered[df_filtered['Sample_Name'] == 'Team1']['Ratio']

# Perform the independent two-sample Student's t-test
# We set equal_var=True because Levene's test indicated equal variances
stat, p_value = ttest_ind(reporter_norm_only_data, team1_data, equal_var=True)

print(f"Student's t-test for Reporter_Norm_only vs. Team1:")
print(f"  t-statistic = {stat:.3f}")
print(f"  p-value = {p_value:.8f}") # show 8 decimal places

alpha = 0.05 # Significance level

if p_value < alpha:
    print(f"There is a statistically significant difference between the means of Reporter_Norm_only and Team1.")
else:
    print(f"There is no statistically significant difference between the means of Reporter_Norm_only and Team1.")
parametric
Welch's t-test
Unequal variance or unequal n
t = x̄₁ − x̄₂ √(s₁²/n₁ + s₂²/n₂)
s² = group variance
df via Welch–Satterthwaite
n = group size

1 : 1  Pairwise — Non-Parametric
non-parametric
Mann-Whitney U
Non-normal data, rank-based
U₁ = n₁n₂ + n₁(n₁+1) 2 − R₁
R₁ = sum of ranks (group 1)
n = group sizes
U = min(U₁, U₂)

parametric
Paired t-test
Normal differences, two matched/repeated measurements
t = d̄ s_d / √n
d̄ = mean of pairwise differences
s_d = std dev of differences
n = number of pairs 
# Paired t-test (before/after or matched pairs)

non-parametric
Wilcoxon Signed-Rank
Non-normal differences, two matched/repeated measurements
W = Σ [ sgn(d) · R(|d|) ]
d = pairwise differences
R(|d|) = rank of |d|
W = sum of signed ranks

Multi-Group Analysis
ANOVA
One-Way ANOVA
3+ groups, equal variance, normal data
F = MSbetween MSwithin
MS = mean square
SS_b = between-group SS
SS_w = within-group SS

post-hoc
Tukey's HSD
Equal n, equal variance; all pairwise comparisons
q = x̄ᵢ − x̄ⱼ √(MSw / n)
q = studentized range stat
MS_w = within-group MS
n = group size 
from statsmodels.stats.multicomp import pairwise_tukeyhsd

post-hoc
Games-Howell
Unequal n or unequal variance
q* = x̄ᵢ − x̄ⱼ √(s²ᵢ/nᵢ + s²ⱼ/nⱼ)
df via Welch-Satterthwaite
s² = per-group variance
critical value from q distribution

non-parametric
Kruskal-Wallis
Non-normal data, 3+ independent groups. Rank-based ANOVA alternative.
H = 12 N(N+1) · Σ Rᵢ² nᵢ − 3(N+1)
Rᵢ = sum of ranks in group i
nᵢ = size of group i
N = total sample size

non-parametric
Dunn's Test
Post-hoc for Kruskal-Wallis; pairwise rank-based comparisons
zij = R̄ᵢ − R̄ⱼ √ N(N+1) 12 · nᵢ + nⱼ nᵢ · nⱼ
R̄ᵢ = mean rank of group i
N = total sample size
nᵢ = size of group i

post-hoc
Bonferroni
Conservative; divide α by # comparisons
α' = α / m
α = desired significance level
m = number of comparisons
reject H₀ if p < α' 
import scikit_posthocs as sp

# Perform Dunn's test with Bonferroni p-value correction
dunn_bonferroni_results = sp.posthoc_dunn(data_for_dunn, p_adjust='bonferroni')

# Label the rows and columns with sample names
dunn_bonferroni_results.columns = sample_names
dunn_bonferroni_results.index = sample_names

def highlight_significant(val):
    return 'background-color: red' if val < 0.05 else ''

print("Dunn's Post-Hoc Test Results (p-values adjusted using Bonferroni):")
display(dunn_bonferroni_results.style.map(highlight_significant))
p-adjustment
Benjamini-Hochberg
FDR control; less conservative than Bonferroni, more statistical power
reject if p(i) ≤ i · q m
p(i) = i-th ranked p-value
q = desired FDR threshold
m = total comparisons 
import scikit_posthocs as sp
