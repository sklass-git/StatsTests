Assumption Check — Normality
normality
Shapiro-Wilk
Best for small samples (n < 50). Most powerful normality test.
W = (Σ aᵢxᵢ)² Σ (xᵢ − x̄)²
aᵢ = order statistic coefficients
xᵢ = ordered sample values
p < 0.05 → reject normality 
# Shapiro-Wilk normality test (best for n < 50)
from scipy import stats
import numpy as np

data = [23, 25, 28, 22, 27, 24, 26, 21, 29, 25]

stat, p_val = stats.shapiro(data)

print(f"W-statistic: {stat:.4f}")
print(f"p-value:     {p_val:.4f}")
if p_val > 0.05:
    print("Fail to reject normality → data is likely normal")
else:
    print("Reject normality → use non-parametric test")
Assumption Check — Variance Equality
variance
Levene's Test
Robust to non-normality. Preferred for real-world data.
W = (N−k) · Σnᵢ(Z̄ᵢ−Z̄)² (k−1) · ΣΣ(Zᵢⱼ−Z̄ᵢ)²
Zᵢⱼ = |xᵢⱼ − x̄ᵢ| (deviations)
k = number of groups
p < 0.05 → unequal variances 
# Levene's test for equality of variances
# Robust to non-normality — use this by default
from scipy import stats

group1 = [23, 25, 28, 22, 27, 24]
group2 = [30, 45, 29, 55, 31, 60]  # high variance
group3 = [35, 33, 36, 34, 37, 35]

stat, p_val = stats.levene(group1, group2, group3)

print(f"W-statistic: {stat:.4f}")
print(f"p-value:     {p_val:.4f}")
if p_val > 0.05:
    print("Equal variances → use Student's t / standard ANOVA")
else:
    print("Unequal variances → use Welch's t / Games-Howell")
variance
Bartlett's Test
More powerful than Levene's, but requires normality.
χ² = (N−k)ln(s²ₚ) − Σ(nᵢ−1)ln(s²ᵢ) c
s²ₚ = pooled variance
c = correction factor
p < 0.05 → unequal variances 
# Bartlett's test — use only when data is normal
from scipy import stats

group1 = [23, 25, 28, 22, 27, 24]
group2 = [30, 45, 29, 55, 31, 60]
group3 = [35, 33, 36, 34, 37, 35]

stat, p_val = stats.bartlett(group1, group2, group3)

print(f"χ²-statistic: {stat:.4f}")
print(f"p-value:      {p_val:.4f}")
if p_val > 0.05:
    print("Equal variances → proceed with parametric tests")
else:
    print("Unequal variances → use Welch's t / Games-Howell")
1 : 1  Pairwise — Parametric
parametric
Student's t-test
Equal variance, two independent groups
t = x̄₁ − x̄₂ sₚ · √(1/n₁ + 1/n₂)
sₚ = pooled std dev
x̄ = group mean
n = group size 
# Student's t-test (equal variance)
import numpy as np
from scipy import stats

# Example data
group1 = [23, 25, 28, 22, 27, 24]
group2 = [30, 32, 29, 35, 31, 33]

# Run test (equal_var=True for Student's)
t_stat, p_val = stats.ttest_ind(group1, group2, equal_var=True)

print(f"t-statistic: {t_stat:.4f}")
print(f"p-value:     {p_val:.4f}")
print(f"Significant: {p_val < 0.05}")
parametric
Welch's t-test
Unequal variance or unequal n
t = x̄₁ − x̄₂ √(s₁²/n₁ + s₂²/n₂)
s² = group variance
df via Welch–Satterthwaite
n = group size 
# Welch's t-test (unequal variance)
from scipy import stats

group1 = [23, 25, 28, 22, 27, 24]
group2 = [30, 32, 29, 35, 31, 33]

# equal_var=False → Welch's (scipy default)
t_stat, p_val = stats.ttest_ind(group1, group2, equal_var=False)

print(f"t-statistic: {t_stat:.4f}")
print(f"p-value:     {p_val:.4f}")
print(f"Significant: {p_val < 0.05}")
1 : 1  Pairwise — Non-Parametric
non-parametric
Mann-Whitney U
Non-normal data, rank-based
U₁ = n₁n₂ + n₁(n₁+1) 2 − R₁
R₁ = sum of ranks (group 1)
n = group sizes
U = min(U₁, U₂) 
# Mann-Whitney U test
from scipy import stats

group1 = [5, 7, 3, 8, 6, 4]
group2 = [9, 11, 10, 13, 12, 14]

u_stat, p_val = stats.mannwhitneyu(
    group1, group2, alternative='two-sided'
)

print(f"U-statistic: {u_stat:.4f}")
print(f"p-value:     {p_val:.4f}")
print(f"Significant: {p_val < 0.05}")
parametric
Paired t-test
Normal differences, two matched/repeated measurements
t = d̄ s_d / √n
d̄ = mean of pairwise differences
s_d = std dev of differences
n = number of pairs 
# Paired t-test (before/after or matched pairs)
from scipy import stats

before = [85, 90, 78, 92, 88, 76]
after  = [88, 95, 80, 94, 91, 79]

t_stat, p_val = stats.ttest_rel(before, after)

print(f"t-statistic: {t_stat:.4f}")
print(f"p-value:     {p_val:.4f}")
print(f"Significant: {p_val < 0.05}")
non-parametric
Wilcoxon Signed-Rank
Non-normal differences, two matched/repeated measurements
W = Σ [ sgn(d) · R(|d|) ]
d = pairwise differences
R(|d|) = rank of |d|
W = sum of signed ranks 
# Wilcoxon Signed-Rank test (non-parametric paired)
from scipy import stats

before = [85, 90, 78, 92, 88, 76]
after  = [88, 95, 80, 94, 91, 79]

w_stat, p_val = stats.wilcoxon(before, after)

print(f"W-statistic: {w_stat:.4f}")
print(f"p-value:     {p_val:.4f}")
print(f"Significant: {p_val < 0.05}")
1 : 2+  Multi-Group — ANOVA
ANOVA
One-Way ANOVA
3+ groups, equal variance, normal data
F = MSbetween MSwithin
MS = mean square
SS_b = between-group SS
SS_w = within-group SS 
# One-Way ANOVA
from scipy import stats

groupA = [20, 22, 19, 21, 23]
groupB = [28, 30, 27, 29, 31]
groupC = [35, 33, 36, 34, 37]

f_stat, p_val = stats.f_oneway(groupA, groupB, groupC)

print(f"F-statistic: {f_stat:.4f}")
print(f"p-value:     {p_val:.4f}")
if p_val < 0.05:
    print("Significant → run post-hoc tests")
else:
    print("Not significant")
non-parametric
Kruskal-Wallis
Non-normal data, 3+ independent groups. Rank-based ANOVA alternative.
H = 12 N(N+1) · ΣRᵢ²/nᵢ − 3(N+1)
Rᵢ = sum of ranks in group i
nᵢ = size of group i
N = total sample size 
# Kruskal-Wallis test (non-parametric, 3+ groups)
from scipy import stats

groupA = [5, 7, 3, 8, 6]
groupB = [12, 15, 11, 14, 13]
groupC = [20, 18, 22, 19, 21]

h_stat, p_val = stats.kruskal(groupA, groupB, groupC)

print(f"H-statistic: {h_stat:.4f}")
print(f"p-value:     {p_val:.4f}")
if p_val < 0.05:
    print("Significant → run post-hoc (e.g. Dunn's test)")
else:
    print("Not significant")
Post-Hoc Analysis  (after significant ANOVA)
post-hoc
Tukey's HSD
Equal n, equal variance; all pairwise comparisons
q = x̄ᵢ − x̄ⱼ √(MSw / n)
q = studentized range stat
MS_w = within-group MS
n = group size 
# Tukey's HSD post-hoc test
import pandas as pd
from statsmodels.stats.multicomp import pairwise_tukeyhsd

# Build a long-format dataframe
data = [20,22,19,21,23, 28,30,27,29,31, 35,33,36,34,37]
groups = ['A']*5 + ['B']*5 + ['C']*5

result = pairwise_tukeyhsd(
    endog=data, groups=groups, alpha=0.05
)
print(result.summary())
post-hoc
Games-Howell
Unequal n or unequal variance
q* = x̄ᵢ − x̄ⱼ √(s²ᵢ/nᵢ + s²ⱼ/nⱼ)
df via Welch-Satterthwaite
s² = per-group variance
critical value from q distribution 
# Games-Howell post-hoc (via pingouin)
# !pip install pingouin
import pandas as pd
import pingouin as pg

df = pd.DataFrame({
    'value':  [20,22,19,21,23, 28,30,27,29,31, 35,33,36,34,37],
    'group':  ['A']*5 + ['B']*5 + ['C']*5
})

result = pg.pairwise_gameshowell(
    data=df, dv='value', between='group'
)
print(result[['A', 'B', 'meanA', 'meanB', 'pval']])
post-hoc
Bonferroni
Conservative; divide α by # comparisons
α' = α / m
α = desired significance level
m = number of comparisons
reject H₀ if p < α' 
# Bonferroni correction on pairwise t-tests
from itertools import combinations
from scipy import stats
from statsmodels.stats.multitest import multipletests

groups = {'A':[20,22,19,21,23], 'B':[28,30,27,29,31], 'C':[35,33,36,34,37]}
pairs = list(combinations(groups.keys(), 2))
p_vals = [stats.ttest_ind(groups[a], groups[b])[1] for a, b in pairs]

reject, p_corr, _, _ = multipletests(p_vals, method='bonferroni')
for (a, b), p, r in zip(pairs, p_corr, reject):
    print(f"{a} vs {b}: p={p:.4f}  reject={r}")
